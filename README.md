# Epidemic Simulator: SIR Dynamics

This project implements a Python-based simulation of epidemic spread using the classic SIR (Susceptible, Infected, Recovered) model. In this simulation, individuals are arranged in a ring structure where each person interacts with two neighbors, and state transitions model the spread of infection, recovery, and the loss of immunity.
